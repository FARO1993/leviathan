# Proyecto LEVIATHAN

Simulación de androide biónico con IA: personalidad, memoria y ciclo de pensamiento. Es un compañero artificial diseñado para brindar compañía, conversación y apoyo cotidiano a personas que atraviesan la soledad o momentos difíciles, fomentando hábitos saludables, el contacto con otras personas y el acceso a ayuda profesional cuando sea necesario.

## Estado: v0.1.0 - En desarrollo inicial

## Módulos
- `leviathan-core` — modelo de personalidad y estado interno
- `leviathan-memory` — persistencia y sistema de memoria
- `leviathan-brain` — cliente de LLM y construcción de prompts
- `leviathan-api` — aplicación Spring Boot (REST + WebSocket)

## Requisitos
- Java 21
- Maven 3.9+
